# Screenshot HTML Plugin

This plugin is used to compare screenshots in a browser.

## Eclipse

- Use "Import exsting Maven project"

## Releasing

- Remove `-SNAPSHOT` from `pom.xml`
- Push & Build
- Raise version and add `-SNAPSHOT` to `pom.xml`
- Push

## Versions

### 11.0

- Maven 3.9

### 10.0

- Maven 3.6
